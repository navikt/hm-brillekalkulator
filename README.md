# hm-brillekalkulator

Åpen kalkulator på nav.no for å gi en indikasjon på om barn kan få støtte til briller og hvilken sats. 
https://www.nav.no/hjelpemidler/brillekalkulator/

### Utvikling 

Kjør: 

```
npm install 
npm run dev
```
